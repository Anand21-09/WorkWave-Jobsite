# WorkWave – Job Portal Backend 🚀

WorkWave is a backend-based Job Portal application built using Spring Boot and MySQL.

The project focuses on building a real-world hiring platform where recruiters can post jobs and candidates can apply for them.

## Currently Implemented ✅
- Job CRUD APIs
- Search Functionality
- Layered Architecture
- Exception Handling
- REST APIs
- MySQL Integration
- JPA/Hibernate

## Tech Stack 🛠️
- Java
- Spring Boot
- Spring Data JPA
- Hibernate
- MySQL
- Maven
